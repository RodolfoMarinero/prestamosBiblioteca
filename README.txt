Instrucciones para la ejecución:

1. Primero, se deben ejecutar los 3 scripts que se encuentran en la carpeta llamada MYSQL. Estos crean la base de datos y las 3 tablas que deben incluirse.
2. La carpeta prestamosBiblioteca contiene mi proyecto completo. En la dirección prestamosBiblioteca/src/main/java/com/mycompany/prestamosbiblioteca se encuentran mis clases .java necesarias para la conexión a la base de datos, así como también mis estructuras y métodos.
3. En la dirección prestamosBiblioteca/src/main/webapp se encuentran mis archivos .jsp, los cuales son mis interfaces y algunos scripts.
4. El archivo principal que debe ejecutarse es "index.jsp", ya que es mi vista principal.
